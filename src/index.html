About Me

I'm currently focused on web development, working primarily with Webpack, jQuery, and TailwindCSS to create responsive, efficient, and well-structured frontends. I enjoy bridging older, battle-tested tools like jQuery with modern workflows, and I put a lot of thought into how things are organized and delivered to the browser. Outside of frontend work, I've used technologies like Firebase to build lightweight backend services, and I've developed and published Python packages in the past. I'm always interested in learning new tools or diving deeper into the ones I already use — especially when it helps me ship cleaner code or streamline development workflows. Whether I'm fine-tuning a build config or wiring up a UI component, I like building things that are both functional and easy to maintain.
